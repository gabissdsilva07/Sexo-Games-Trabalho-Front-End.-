O que é HTTP?

É um protocolo de comunicação usado para transferir dados entre um cliente (geralmente um navegador) e um servidor na web.

O que é HTTPS?

É a versão segura do HTTP, onde os dados são criptografados para proteger a privacidade e segurança das informações trocadas.

O que é um Cliente?

É o dispositivo ou programa que solicita e recebe dados de um servidor, como um navegador de internet.

O que é um Servidor?

É o sistema que armazena e envia os dados solicitados pelo cliente, como um site, banco de dados ou arquivo.
